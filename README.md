# Dotfiles for Hyprland and Wayland

My personal dotfiles for Hyprland-based setups. Includes configuration for:

- Hyprland
- hyprlock
- hyprpaper
- hypridle
- starship
- Waybar
- swaync
- Wofi
- Shell (`.bashrc` / `.zshrc`)
- wallpaper

## 🔧 Setup

Clone the repo:

```bash
git clone https://github.com/NicholasAngel/hyprdotfiles.git ~/dotfiles
cd ~/dotfiles
bash setup.sh
```
